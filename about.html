This application was designed to help Bible students to memorize versicles.
I used these tools to create it:

  • Bootstrap Libraries
  • W3school tutorials
  • Icons8 free favicons
  • Powerpoint free Images of Bibles
  • Javascript Code for the application

I used the Bootstrap library version 3.4.1 where we can find the template "Marketing"
I modified and adapt it to my application. Here you have the referenece.

The JavaScript code read the text letter by letter and once it finds a space, it determines is a whole word.
For the first exercise, if the word is equal or longer than 6 letters (>=6), then it replaces with blanks ("_").
For the second exercise, if theword is equal or longer that 3 letters (>=3), then it replaces with blanks ("_").
These word lengths can be modified if needed.
